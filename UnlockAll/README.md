# UnlockAll
	- Unlocks every locked thing
	- Keeps profile intact
	- Supports [HjUpdaterAPI](https://thunderstore.io/package/Lodington/HjUpdaterAPI/) as soft dependency for auto updating 

	
## Installation
Drop `UnlockAll.dll` into `\BepInEx\plugins\`
