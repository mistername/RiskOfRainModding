# BalancedObliterate
	- Gives amount of stages times difficulty multiplier as coins on run end
	- Difficulty multiplier is configurable in `BepInEx\config\BalancedObliterate.cfg`

## Installation
Drop `BalancedObliterate.dll` into `\BepInEx\plugins\`
